## 🚀 Fitur Utama

### 1. **Dashboard**
- Tampilan saldo real-time
- Quick access ke semua fitur
- Promo banner dinamis
- Riwayat transaksi terakhir

### 2. **Bayar Tagihan**
- **PLN (Listrik)**: Cek dan bayar tagihan listrik
- **PDAM (Air)**: Pembayaran tagihan air
- **Internet**: Bayar tagihan internet (IndiHome, MyRepublic, Biznet)
- **Seminar/Event**: Pembayaran seminar dan workshop

### 3. **SPP / Biaya Kuliah**
- Cek tagihan berdasarkan NIM
- Pilih multiple tagihan sekaligus
- Status lunas/belum per item
- Total otomatis dihitung

### 4. **Isi Pulsa & Paket Data**
- 6 provider: Telkomsel, XL, Indosat, Tri, Smartfren, Axis
- Nominal pulsa: Rp10.000 - Rp200.000
- Paket data: 1GB - 12GB
- Deteksi provider otomatis dari nomor HP

### 5. **Riwayat Transaksi**
- Filter berdasarkan status
- Statistik pengeluaran chart
- Hapus semua riwayat
- Print bukti pembayaran

### 6. **Metode Pembayaran**
- ✅ Virtual Account (BCA, BNI, Mandiri, BRI, BTN)
- ✅ QRIS (dengan QR Code generator)
- ✅ Teller / Kasir (dengan kode pembayaran)

### 7. **Fitur Lainnya**
- 🌙 Dark Mode
- 💾 Penyimpanan lokal (localStorage)
- 📄 Print struk pembayaran
- 🔔 Toast notification
- 📱 Mobile-first design

## 🛠 Teknologi

### Frontend
- **HTML5** - Struktur aplikasi
- **CSS3** - Styling dengan tema hijau/teal
- **JavaScript (ES6)** - Logika aplikasi
- **Bootstrap 5** - Komponen UI (minimal)
- **Font Awesome 6** - Ikon

### Library Tambahan
- **QRCode.js** - Generator QR Code
- **Chart.js** - Grafik statistik
- **jsPDF** - Export PDF (opsional)
